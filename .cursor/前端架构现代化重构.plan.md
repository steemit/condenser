## 项目现状分析

当前项目使用：

- React 15.6.2 + React Router 3.2.0
- Redux 3.7.2 + Redux-Saga 0.16.0 + Immutable.js
- Webpack 3.7.1 + Babel 6
- Koa 1.4.1 (自定义 SSR)
- SCSS + Foundation Sites

## 目标架构

- **React 19**
- **Next.js 16**（App Router，内置 SSR；实际锁定版本见 `package.json`）
- **Redux Toolkit**（新栈无 redux-saga 依赖；复杂异步以 thunk / Route Handlers 等为主）
- **TypeScript**（逐步迁移）
- **Tailwind CSS v4** + **shadcn/ui**（base-nova，替代 Foundation 为新 UI 主路径）
